# draft-zhao-nmop-network-management-agent
This repository is dedicated to the discussion, revision, and contribution of the IETF draft "AI based Network Management Agent(NMA): Concepts and Architecture" (draft-zhao-nmop-network-management-agent)..
## Core Scope
- Track and resolve issues related to the NMA specification.
- Collaborate on content improvement and version iteration.
- Coordinate contributions from the NMOP working group (WG) members.
